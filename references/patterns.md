# 常见模式与修改方向

把这份清单当成复查线索，不要把它变成禁词表。单独出现一次通常不构成问题；当多个模式反复出现、掩盖事实或破坏节奏时再改。

| 模式 | 修改方向 |
|---|---|
| “这是一个很好的问题。下面我将……” | 直接回答问题。 |
| “在当今快速发展的……时代” | 写出具体时间、产品或情境。 |
| “不仅……而且……；不是……而是……” | 直接陈述核心判断和边界。 |
| “意义重大、影响深远、发挥关键作用” | 写清具体影响；没有信息就删除。 |
| “综上所述、总而言之、值得注意的是” | 直接进入结论或下一条事实。 |
| 连续堆叠三个近义词 | 保留最准确的一个。 |
| “可能会带来一定影响” | 改为“可能影响……”或说明不确定性来自哪里。 |
| 抽象主体，如“数据告诉我们” | 已知时写出数据来源、研究者或决策者。 |
| 每节末尾都重复小结 | 只在帮助读者判断时保留一次结论。 |
| 标题只重复主题 | 让标题说明这一节解决什么问题。 |
| 每段长度和句式高度一致 | 按意思自然分段，混合长短句，但不要故意制造混乱。 |
| 无来源的“业内人士认为” | 给出来源；无法确认就删除或明确不确定性。 |
| 过度加粗、表格、Emoji 和分隔线 | 只保留能帮助导航、比较或提醒风险的格式。 |
| 结尾重复全文并附泛化展望 | 用具体决定、限制或下一步收尾；不需要时直接结束。 |

## 不要误改

- 不改代码、URL、引用、专有名词和精确数字。
- 不为了“像人”而添加虚构经历、情绪、俚语、错别字或强硬观点。
- 不为了句式变化替换稳定的技术术语。
- 不承诺通过任何 AI 检测器；检测结果会受模型、语种、文本长度和检测器版本影响。
